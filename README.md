# smart_medicine 项目简介

这是一个基于 SpringBoot 开发的 Java Web 项目，旨在提供智慧医药服务。

## 角色介绍

系统共设计了三种角色，权限逐级递增：

| 角色       | 主要权限                                                     |
| ---------- | ------------------------------------------------------------ |
| **游客**   | 登录注册、疾病与药品搜索及详情查询。                         |
| **用户**   | 除游客权限外，还包括个人资料修改、密码修改、意见反馈、智能医生咨询。 |
| **管理员** | 除用户权限外，还包括疾病管理、药品管理、反馈管理。           |

## 功能介绍

### 游客功能

*   **登录注册**: 注册成为系统用户。
*   **系统主页**: 浏览系统主页，搜索疾病、药品信息并查看详情。

### 用户功能

*   **登录注册**: 注册账号（邮件验证码）、使用账号密码登录。
*   **个人资料**: 修改个人资料（姓名、年龄、手机号、头像等）、修改登录密码。
*   **系统反馈**: 提交系统反馈意见。
*   **智能医生**: 与智能医生进行交流咨询。

### 管理员功能

*   **登录注册**: 注册账号（邮件验证码）、使用账号密码登录。
*   **个人资料**: 修改个人资料（姓名、年龄、手机号、头像等）、修改登录密码。
*   **系统反馈**: 提交系统反馈意见。
*   **智能医生**: 与智能医生进行交流咨询。
*   **疾病管理**: 发布、编辑（名称、原因、症状、分类等）、删除疾病信息。
*   **药品管理**: 发布、编辑（名称、关键词、功效、用法用量、类型等）、关联疾病、删除药品信息。
*   **反馈管理**: 管理用户提交的反馈信息。

## 技术栈

### 前端

*   **HTML、CSS**: 网页内容与样式设计。
*   **JavaScript、JQuery**: 动态功能与交互。
*   **Bootstrap**: 响应式前端开发框架。
*   **前端模板**: 智慧医药系统页面模板。

### 后端

*   **SpringBoot**: 简化 Spring 应用开发。
*   **SpringMVC**: 构建 Web 应用程序的 MVC 模块。
*   **MyBatis**: Java 持久化框架，简化数据库操作。
*   **Thymeleaf**: Java 模板引擎，用于渲染 HTML 内容。
*   **Druid**: 高效数据查询系统，用于时序数据聚合查询。

## 开发工具与环境

*   **开发工具**: IntelliJ IDEA (含 Lombok 插件), WebStorm, Navicat。
*   **开发环境**: JDK 1.8, Maven 3.6, MySQL 5.7。
*   **云服务**: 阿里云 OSS 对象存储 (用于存储二进制文件), 阿里云 AIGC 通义千问大语言模型 (用于智能医生咨询)。

## 项目结构

本项目采用标准 SSM 目录结构和 MVC 三层架构：

*   `src/main/java/xxx/component`: Java 常用组件 (如登录拦截器)。
*   `src/main/java/xxx/controller`: 控制层。
*   `src/main/java/xxx/entity`: 实体类 (模型)。
*   `src/main/java/xxx/dao`: MyBatis 映射接口 (数据访问层)。
*   `src/main/java/xxx/service`: 业务逻辑抽象层。
*   `src/main/java/xxx/utils`: 工具类。
*   `src/main/resources/smart_medicine.sql`: 数据库脚本。
*   `src/main/resources/static`: 前端静态资源。
*   `src/main/resources/templates`: 前端页面。
*   `src/main/resources/application.yml`: SpringBoot 配置文件。
*   `pom.xml`: Maven 项目依赖管理文件。
