> **项目简介**：随着医疗信息化与人工智能技术的深度融合，智慧医疗已成为解决医疗资源分布不均、提升诊断效率的重要路径。本项目设计并实现了一套基于 **Spring Boot** 的医疗咨询和自我诊断健康中心系统，旨在通过 **AIGC 技术** 为医疗咨询智能化提供新可能。

---

## 🌟 核心功能 (Core Features)

| 角色         | 功能模块        | 详细描述                                                  |
| :----------- | :-------------- | :-------------------------------------------------------- |
| **👤 用户端** | **AI 智能医生** | 基于通义千问大模型 API 实现智能交互的多轮对话咨询。       |
|              | **疾病百科**    | 支持疾病与药品的关联查询，提供详尽的健康知识。            |
|              | **药品查询**    | 快速检索药品信息，辅助自我诊断。                          |
| **⚙️ 管理员** | **运营管理**    | 疾病、药品、医生信息的全生命周期管理。                    |
|              | **数据看板**    | 基于 **ECharts** 实现的数据可视化大屏，实时掌握运营动态。 |
|              | **反馈处理**    | 集中处理用户反馈，优化服务体验。                          |
| **🌐 游客**   | **公共浏览**    | 浏览医疗资讯、疾病百科，支持快速注册与登录。              |

## 🛠️ 技术架构 (Tech Stack)

### 后端技术

- **核心框架**：Spring Boot (轻量化设计，易于部署)
- **持久层**：MyBatis (高效的数据持久化方案)
- **云服务**：阿里云 OSS (存储用户头像等二进制数据)
- **AI 引擎**：通义千问大模型 API (驱动智能医生咨询)

### 前端技术

- **模板引擎**：Thymeleaf
- **响应式框架**：Bootstrap (适配多端访问)
- **可视化**：ECharts (专业数据大屏展示)

---

## 🔒 安全保障 (Security)

系统全方位保障用户信息与数据安全：

- ✅ **身份验证**：集成邮箱验证码机制，确保用户身份真实性。
- ✅ **密码安全**：结合 **BCrypt** 强哈希加密算法存储用户密码。
- ✅ **传输加密**：采用 **HTTPS** 协议，有效避免数据在传输过程中的泄露。
- ✅ **权限控制**：严格的后端权限校验，防止未授权访问。

## 🚀 项目总结
 
本项目具备**轻量化设计**与**强实用性**，能够为中小型医疗机构提供低成本的信息化工具。通过疾病与药品的关联查询、智能交互的多轮对话以及高效的数据存储，系统全面满足了用户自诊咨询与管理员综合管理的多样化需求，有力推动了医疗信息服务的数字化升级。
