    随着医疗信息化与人工智能技术的深度融合，智慧医疗已成为解决医疗资源分布不均、提升诊断效率的重要路径。当前医疗系统存在信息碎片化、交互体验不足等问题，而AIGC技术的发展为医疗咨询智能化提供了新可能。针对上述医疗咨询与自我诊断服务中的实际痛点，本文设计并实现一套基于Spring Boot的医疗咨询和自我诊断健康中心系统。
    系统采用前后端分离架构设计，面向管理员、用户、游客三类核心角色提供服务。管理员可进行疾病与药品管理、医生管理、用户反馈处理、数据统计等整体运营管理；用户端支持AI智能医生咨询、疾病百科查询、药品信息查询、个人信息管理等功能；游客可浏览公共信息并进行注册登录。
    技术上，后端采用Spring Boot整合MyBatis实现数据持久化，使用阿里云OSS存储用户头像等二进制数据，通过通义千问大模型API实现智能医生咨询功能；前端基于Thymeleaf模板引擎与Bootstrap响应式框架搭建页面，并使用ECharts实现数据可视化大屏展示。系统支持疾病与药品的关联查询、智能交互的多轮对话、数据的高效存储与访问，可满足用户自诊咨询与管理员综合管理的多样化需求。
    在安全方面，系统通过邮箱验证码验证用户身份，结合BCrypt加密保障用户密码安全，采用HTTPS协议确保数据传输安全，有效避免未授权访问与信息泄露。整体架构具备轻量化设计特点与较强的实用性，能够为中小型医疗机构提供低成本信息化工具，推动医疗信息服务的数字化升级
